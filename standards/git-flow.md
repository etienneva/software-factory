# Git Flow

## Branches

main

production uniquement

develop

intégration

feature/*

développement

hotfix/*

corrections urgentes

## Workflow

Issue
→ Feature Branch
→ Pull Request
→ Review
→ Merge develop
→ Validation
→ Merge main

## Pull Request

Doit contenir :

* objectif
* changements réalisés
* impacts
* tests effectués

## Interdictions

* commit direct sur main
* merge sans review
